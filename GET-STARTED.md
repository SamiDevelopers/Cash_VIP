## 🔌 APIs المتاحة

### 15 API Endpoint جاهز للاستخدام

**Authentication:**
```
POST /api/auth/register      - تسجيل
POST /api/auth/login         - دخول
```

**Apps:**
```
GET /api/apps                - جميع
GET /api/apps/:id            - واحد
POST /api/apps               - إضافة (Admin)
PUT /api/apps/:id            - تعديل (Admin)
DELETE /api/apps/:id         - حذف (Admin)
```

**Transactions:**
```
GET /api/transactions            - جميع
POST /api/transactions           - إضافة
GET /api/transactions/stats/dashboard - إحصائيات
```

**Offers:**
```
GET /api/offers                  - جميع
GET /api/offers/active           - النشط
PUT /api/offers/:id/activate     - تفعيل (Admin)
PUT /api/offers/:id              - تحديث (Admin)
```
