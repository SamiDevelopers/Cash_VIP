# Quick Start Guide - دليل البدء السريع

## 🚀 خطوات التشغيل الفوري

### 1️⃣ تثبيت المتطلبات
```bash
cd backend
npm install
```

### 2️⃣ تشغيل الخادم
```bash
npm run dev
```

### 3️⃣ فتح الواجهة
- اذهب إلى `frontend/index.html`
- أو افتحه في المتصفح مباشرة

### 4️⃣ بيانات الاختبار
```

   Username: admin
```

## 💡 نصائح الاستخدام

### للموظف:
1. اختر التطبيق من القائمة
2. أدخل المبلغ
3. ستحصل على حسابات فورية
4. أرسل للعميل عبر واتساب
5. احفظ البيانات

### للمدير:
1. اذهب إلى "الإعدادات"
2. أضف أو عدّل التطبيقات
3. فعّل العروض الخاصة
4. شاهد التقارير المفصلة
