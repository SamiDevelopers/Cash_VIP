# 📋 دليل الاستخدام المتقدم - Cash VIP System

## 🎯 نظرة عامة على النظام

نظام متكامل لتسييل تطبيقات التقسيط يجمع بين:
- **Backend قوي**: Node.js + Express.js + SQLite
- **Frontend حديث**: HTML5 + Tailwind CSS + Vanilla JavaScript
- **أمان متقدم**: JWT Authentication + Role-Based Access Control
- **تجربة مستخدم ممتازة**: واجهة عربية كاملة RTL

---

## 📊 مكونات النظام

### 🔧 Backend (خادم التطبيق)

#### 📁 هيكل الملفات
```
backend/
├── server.js              # نقطة البداية - Express.js
├── database.js            # إدارة قاعدة البيانات
├── package.json           # Dependencies
├── .env                   # متغيرات البيئة
├── middleware/
│   └── auth.js            # JWT Middleware
└── routes/
    ├── auth.js            # /api/auth
    ├── apps.js            # /api/apps
    ├── transactions.js    # /api/transactions
    └── offers.js          # /api/offers
```

#### 🔌 API Endpoints

**Authentication:**
```
POST   /api/auth/register     - تسجيل مستخدم جديد
POST   /api/auth/login        - تسجيل الدخول
```

**Apps Management:**
```
GET    /api/apps             - جميع التطبيقات
GET    /api/apps/:id         - تطبيق واحد
POST   /api/apps             - إضافة (Admin فقط)
PUT    /api/apps/:id         - تعديل (Admin فقط)
DELETE /api/apps/:id         - حذف (Admin فقط)
```

**Transactions:**
```
GET    /api/transactions           - جميع العمليات
POST   /api/transactions           - إضافة عملية
GET    /api/transactions/stats/dashboard - إحصائيات
```

**Offers:**
```
GET    /api/offers                    - جميع العروض
GET    /api/offers/active             - العرض النشط
PUT    /api/offers/:id/activate       - تفعيل عرض
PUT    /api/offers/:id                - تحديث عرض
```

## 🧮 معادلات الحسابات

### معادلة التسييل الأساسية

```
المبلغ المستقطع = المبلغ الأصلي × (|نسبة التسييل| / 100)

المبلغ المستقبل = المبلغ الأصلي - المبلغ المستقطع

نسبة الخصم الإجمالية = النسبة الأساسية + نسبة العميل + النسبة المحاسبية + النسبة الإدارية
```

### معادلات الأقساط والفوائد

```
مدة التقسيط بالسنوات = مدة التقسيط بالأشهر ÷ 12

إجمالي الفائدة = المبلغ الأصلي × (الفائدة السنوية ÷ 100) × مدة التقسيط بالسنوات

إجمالي المبلغ = المبلغ الأصلي + إجمالي الفائدة

القسط الشهري = إجمالي المبلغ ÷ مدة التقسيط بالأشهر
```

### معادلات العروض

**Zero Interest Offer:**
```
الفائدة السنوية = 0%
→ الفائدة الإجمالية = 0
→ القسط الشهري = المبلغ ÷ عدد الأشهر
```

**Triple Zero Offer:**
```
الفائدة السنوية = 0%
النسبة الإدارية = 0%
→ المبلغ المستقبل = المبلغ الأصلي
→ الفائدة الإجمالية = 0
```

**Balance Boost Offer:**
```
المبلغ المعدل = المبلغ الأصلي × (1 + نسبة الزيادة ÷ 100)
الفائدة الإجمالية = المبلغ المعدل × (الفائدة ÷ 100) × السنوات
```
